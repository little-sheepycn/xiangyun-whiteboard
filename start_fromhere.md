# 千里之行，始于硬件

折腾的第一步，就是了解教室里的设备

## 教室里有以下设备

- HISENSE白板与附带的电脑[^1]
- 一台视频展台[^2]
- 一根断掉的USB延长线[^3]
- 小米键鼠套装[^4][^5]

 [^1]: 前置USB全部报废，下面有很多USB接口，可以自行购买USB扩展坞以方便使用
 [^2]: 分辨率和刷新速度都堪忧的那种
 [^3]: 需要找人更换，也许需要自费购买
 [^4]: 接收器安装在下方USB，由于金属的电磁屏蔽特性信号极其差，购买拓展坞后建议将接收器移至拓展坞上
 [^5]: __不防水！！！__
